# 图片素材说明

当前页面只保留与文案语义对应的本地素材，避免出现“商品名称和图片不一致”的问题。

## 当前使用原则

- 商品图必须直接对应商品名称，例如“竖向起身扶手”使用竖向扶手图，“门槛坡化垫”使用门槛缓坡图。
- 社区图必须对应帖子内容，例如卫生间复扫帖使用卫生间改造后和扶手图，玄关帖使用线缆整理和上门沟通图。
- 网络公开图可以作为候选，但如果画面和文案不一致，即使是真实图片也不进入页面。

## 后续替换建议

- 设计组替换图片时，保持同名文件即可，不需要改代码。
- 商城图建议统一为浅色真实家居场景、无品牌水印、主体清楚。
- 社区图可以保留“用户随手拍”的真实感，但必须看得出帖子里提到的改造对象。
